\subsection{Prerequisites}
\begin{itemize}
  \item Rust toolchain (\texttt{cargo}) for building or using the crate.
  \item Python 3 in \texttt{PATH} for rendering \texttt{.v} and \texttt{.inst} from generated Python.
\end{itemize}

PyTV invokes \texttt{python3} on Linux/macOS and \texttt{python} on Windows.

\subsection{Install as a Rust Library}
Add PyTV to an existing Rust project:
\begin{verbatim}
cargo add pytv
\end{verbatim}

This updates \texttt{Cargo.toml} with a dependency entry similar to:
\begin{verbatim}
[dependencies]
pytv = "0.5.6"
\end{verbatim}

Default crate features include \texttt{inst} and \texttt{macro}.
If needed, enable or disable them explicitly in \texttt{Cargo.toml}.

\subsection{Install as a Binary Tool}
Install from crates.io:
\begin{verbatim}
cargo install pytv
\end{verbatim}

Build from source:
\begin{verbatim}
cargo build --release
\end{verbatim}
The compiled binary is available at \texttt{target/release/pytv}.

\subsection{Validate the Installation}
After installation, verify command-line availability:
\begin{verbatim}
pytv --help
\end{verbatim}

\subsection{Build This Documentation}
From the \texttt{docs/} directory:
\begin{verbatim}
latexmk -pdf pytv-docs.tex
\end{verbatim}
