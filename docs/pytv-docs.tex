\begin{abstract}
  PyTV (Python Templated Verilog)%
  \footnote{This is the documentation for PyTV version \texttt{\pytvversion{}}.
  Latest digital version of this document is available at \url{https://go.wqzhao.org/pytv-docs}.}
  is a Rust library and binary tool that extends Verilog with Python templating.
  By leveraging Python's flexibility,
  PyTV allows users to write Verilog templates with Python control constructs,
  including \texttt{if} statements, \texttt{for} loops, and function definitions.
  PyTV also supports instantiation of modules with complex parameters
  and enables hierarchy extraction.
\end{abstract}
